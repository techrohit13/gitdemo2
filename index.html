Now transfer local directoris to github Repo

        Step 1 : Initialize git
            git init

        Step 2 : Check Intialization is done by vhecking .git file
            ls -Hidden
        
        Step 3 : Add
            git add . or git add index.html
        
        Step 4 : Commit
            git commit -m "Index File added"

        Step 5 : Link to your github repo
            git remot add origin https://github.com/techrohit13/gitdemo2.git  
        
        Step 6 : Check your setup
            git -v 

        Step 7 : Push to github
            git push -u origin main
